关系型数据模型

Table/Relation

  • 列(Column)
  • 行(Row)

通常是一个瘦长的表

概念

Schema vs. Instance

  • Schema: 类型
    一个表的类型是由每个列的类型决定的

  • Instance： 具体取值
    具体存储哪些记录，每个列的具体指
    由具体用用决定

Key

  • Primary key
  • Foreign key
    是另一个表的主键

主要关系运算

  • Selection（选择）
    从一个表中提取一些行
  • Projection(投影)
    从一个表中提取一些列
  • Join（连接）
    • Equi-join(等值连接)
